Week 2

    In Class

  • Intro to GitHub

  • Intro to Bootstrap

    Homework

  • Download Bootstrap and get yourself set up to begin experimenting with it. I don't expect all of you to fully grasp how everything works immediately, but I would like you to begin exploring the various CSS components. The most important of these is going to be the grid, as this will enable you to create responsive layouts.

  • I will be fielding questions in class and you will have time to experiment with things while I am there, so don't get too caught up on the assignment this week. I just want you all to be actively investigating the framework so things make a bit of sense when we cover them together.

    Homework

  • Continue to explore the Bootstrap grid by creating interactive wireframes. You should focus on the content and layout of your site, and you should employ minimal CSS. Be sure to use responsive measurements (ems, rems, and percents) when relevant. Think mobile first.
